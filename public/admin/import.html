CSV Format Requirements:

  • Required columns: lot_number, title, starting_bid
  • Optional columns: description, category, condition, location, reserve_price, buy_now_price
  • First row must contain column headers
  • UTF-8 encoding recommended

Filename Patterns:

  • lot12-1.jpg, lot12-2.jpg → Lot 12, photos 1 and 2
  • 12_1.jpg, 12_2.jpg → Lot 12, photos 1 and 2
  • item-12-001.jpg → Lot 12, photo 1
